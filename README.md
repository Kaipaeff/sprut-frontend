# Личный кабинет — Визуализация данных мышечной активности

Фронтенд для визуализации данных мышечной активности (React 18, TypeScript, Vite, Recharts).

## Экраны

- **Личный кабинет (главная)** (`/`) — форма загрузки файла с данными мышечной активности (.xlsx), drag-and-drop, индикатор прогресса
- **Визуализация** (`/chart/:id`) — отображение названия файла, интерактивный график зависимости мышечной активности от времени, навигация по графику
- **Список датасетов** (`/datasets`) — GET /api/datasets, список всех загруженных датасетов
- **Редактирование** (`/dataset/:id/edit`) — PUT /api/dataset/:id, форма редактирования датасета

## Запуск

1. Убедитесь, что бэкенд API запущен (например, `http://localhost:8000`).
2. Скопируйте `.env.example` в `.env` и при необходимости измените `VITE_API_URL`.
3. Установка и запуск:

```bash
npm install
npm run dev
```

Сборка: `npm run build`. Превью продакшена: `npm run preview`.

## Стек

- React 18, TypeScript, React Router DOM, Context API, Axios, Recharts, Vite, CSS Modules.

## Ограничения API

- Максимальный размер загрузки файла: 16 MB.
- Формат .xlsx: обязательные столбцы `timestamp`, `emg1`, `emg2`, `emg3`, `emg4`, `angle` (все числа).
